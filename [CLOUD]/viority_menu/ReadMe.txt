    Viority Menu Instructions:

1.  ViorityCore edits:
    In order to get Statements fully working, you will have edit the "ViorityCore" > "settings/ViorityCore.lua" file.
    Following the following steps carefully!

    Viority_License > Put your License here (Check it on our Discord with the command "/license")
    
    and edit all functions you want!

	
2.  Insert the SQL:
    Insert the SQL File in your Database. 


3.  Upload the script:
    Insert the script in your ressource folder.


4.  Start the script in CFG file:
    Now open your CFG file with TxAdmin or an editor and add the line "start viority_menu".


5.  Restart your server.
    Now restart your server and the script will automatically start if you do step 3 carefully.
    Connect to your server and the menu will be visible if you press ESC.

6.  Commands: 
    
    - "event" to add, delete or show all or a specific event per console.
    - "menu_info" see all important information about the script and your server. This is used for support cases!
